# 🇺🇸 U.S. Citizenship Exam Study App

Interactive study tool with 128 questions for the U.S. Citizenship (Naturalization) Test, featuring historical context for 90+ questions.

## Features
- 📖 **Study Mode** — Flashcards with answer reveal and progress tracking
- 🧠 **Practice Test** — Configurable quizzes with scoring (60% to pass)
- 📜 **Historical Context** — Deep background info on 90+ questions
- 📊 **Progress Tracking** — Saved locally in your browser
- 🏷️ **9 Categories** — Filter by topic area

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Deploy to Vercel (free)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click "New Project" → Import your repo
4. Click "Deploy" — done! You'll get a public URL to share.

## Deploy to Netlify (free)

1. Push this folder to a GitHub repository
2. Go to [netlify.com](https://netlify.com) and sign in with GitHub
3. Click "Add new site" → "Import an existing project"
4. Select your repo → Deploy. You'll get a public URL.

## Deploy to CodeSandbox (instant, no GitHub needed)

1. Go to [codesandbox.io](https://codesandbox.io)
2. Create a new Sandbox → Choose "Vite + React"
3. Replace the files with the ones in this project
4. Share the sandbox URL directly

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder, ready to host anywhere.
